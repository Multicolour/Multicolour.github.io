Multicolour examples

Multicolour tries to be as helpful as possible with creating and maintaining your REST APIs.

Getting started is really simple and quick, simply start by running

          
            npm install -g multicolour
            multicolour init .

Once you've answered a few simple questions about your project you'll have a functional REST API with some dummy content ready to start

node app.js
