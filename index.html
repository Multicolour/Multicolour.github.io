Multicolour is an open source NodeJS tool that generates a REST API, doing as much of the repetitive work for you so that you get straight to the real problems for your clients. All you need to do is write JSON.

Try the example Multicolour app today.

Getting started couldn't be easier

npm install -g multicolour
+ 
multicolour init .
+ 
node app.js

REST API generation that's easy and fast enough to power your apps, games and websites with no effort.

Self documenting via Swagger UI, OAuth support for popular services like Facebook, Twitter, LinkedIn, Nest and many more. JSON API support to prevent bikeshedding.

CLI tools to help get you started, add new features and keep your app up and running.

Start any project even faster with the init command and a wizard, even less coding to do even more.

Plugins, for everything! extending Multicolour is fast and easy using plugins with no special syntax, just .use your code!

Multicolour's plugin system is designed with ease and flexibility in mind, with only one requirement you can make Multicolour anything you want it to be.
